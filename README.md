quarts
======

quantitative research and trading system

a system aim to support  us equity tick-by-tick intraday models
The whole system includes
- a model in python
- a python research environment to design, debug and quick test-run of the model
- a C++ application hosting python models
- back test for models in C++ environment
- forward test for models in C++ environment
- live trading for models in C++ environment
